# ChallengeConversor

## Descripción

**ChallengeConversor** es un proyecto desarrollado como parte del programa One Oracle Next Education, especializado en backend con Java y Spring. Este desafío se centra en la creación de un conversor de divisas, permitiendo a los usuarios realizar conversiones entre diferentes monedas utilizando una API externa para obtener tasas de cambio actualizadas.

## Funcionalidades

- **Conversión de Divisas**: Permite convertir entre varias monedas, incluyendo EUR, CLP, USD y AUD.
- **Interfaz de Línea de Comandos**: Los usuarios pueden interactuar con el conversor a través de la consola, eligiendo la opción de conversión y especificando la cantidad a convertir.
- **Uso de API Externa**: Utiliza la API de Exchange Rate para obtener tasas de cambio en tiempo real y realizar conversiones precisas.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación utilizado para desarrollar la lógica del backend.
- **Gson**: Biblioteca utilizada para manejar la serialización y deserialización de datos en formato JSON.

## Requisitos

- JDK 11 o superior
- Maven para manejar las dependencias del proyecto
